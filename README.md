# 矿堆库存估值小程序

这是一个原生微信小程序原型，可直接用微信开发者工具导入当前目录测试。

## 测试方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本文件所在目录。
4. 编译运行。

## 当前功能

- 首页为“我的矿堆”，展示矿堆数和预计货值。
- 底部包含“我的矿堆 / 个人中心”两栏导航。
- 右下角新增矿堆。
- 新增矿堆优先填写干重；湿重、水分、收矿来源、堆放地点、备注均为可选。
- 新增矿堆可填写购入价格，库存卡片显示购入价、预计货值和利润。
- 化验单只填写金属/氧化物、品位和品位单位。
- Au、Ag 等稀贵金属自动使用 `g/t`，普通金属默认 `%`，也可手动修改。
- 推荐元素只展示常用项，其他元素可直接手动输入。
- 首页矿堆支持多选，选中后显示“出售”按钮。
- 出售核算页使用宽表展示：矿堆、金属种类、湿重、水分、干重、品位、含量、出售单价和总金额。
- 普通金属按含量 T 计价；Au、Ag 等稀贵金属按含量 g 计价。
- 出售核算页显示购入总价、出售总价和总利润。
- 点击“已完成出售”后，选中的矿堆会从“我的矿堆”移除，并写入个人中心出售记录。
- 个人中心微信登录后可查看往期出售矿堆、往来交易次数和往来买家。
- 往期出售记录可点击查看详情。
- 出售记录支持恢复到库存，防止误触出售。
- 出售记录支持删除，便于清理填错记录。
- 买家和交易单价会保存在用户本地缓存，下次出售时可继续使用。

## 计算口径

```text
优先使用用户填写的干重。

如果未填写干重：
干重 = 湿重 * (1 - 水分%)

品位为 %：
kg/T = 品位 * 10

品位为 g/t：
kg/T = 品位 / 1000

普通金属含量 T = 干重 t * 品位% / 100

稀贵金属含量 g = 干重 t * 品位 g/t

普通金属总价 = 含量 T * 出售单价

稀贵金属总价 = 含量 g * 出售单价

利润 = 出售总价 - 购入总价
```
