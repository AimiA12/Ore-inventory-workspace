# 我的矿堆 Web App

这是浏览器版本的矿堆库存管理 Web App，当前为纯前端本地存储版。

## 打开方式

直接用浏览器打开 `web/index.html` 即可运行。

## 当前功能

- 桌面端库存工作台
- 我的矿堆库存列表
- 新增矿堆、化验单、购入价、收矿来源、堆放地点
- 多选矿堆、删除矿堆、出售核算
- 普通金属按 `%` 计算含量 T，Au / Ag 等稀有金属按 `g/t` 计算含量 g
- 出售完成后移入个人中心的往期出售记录
- 出售记录可查看详情、恢复到库存、删除
- 账号密码登录原型
- 数据保存在浏览器 `localStorage`

## 后续接后端

后续做正式 Web App 时，把 `web/app.js` 中的登录、库存、出售记录读写替换为接口请求即可。
